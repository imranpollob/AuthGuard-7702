# Overleaf Build Instructions

1. Upload all files while preserving the directory structure.
2. Set `main.tex` as the main document.
3. Use pdfLaTeX unless the uploaded project reports a TikZ or font compatibility issue.
4. Run BibTeX through Overleaf's normal build process.
5. Verify that the complete paper, including references and floats, does not exceed eight pages.
6. Inspect every figure at its rendered one- or two-column size for readable labels and grayscale distinction.
7. Inspect and resolve overfull boxes without changing IEEE margins or using negative spacing.
8. Inspect reference formatting, missing fields, and citation order.
9. Recheck the generated PDF metadata for identifying information before submission.
